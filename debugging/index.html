Debugging

To facilitate troubleshooting, you can select the log level you want to log in the Device Settings. Default is set to none. Debug logs almost anything happening while error or warning log only when an error occurs or when something needs your attention. All logs are saved into a csv file.

Accessing Logs​

  1. Navigate to the "Log" page in the WebUI.
  2. View the log entries directly on the page.
  3. Use the available options to either:
    • Delete the log file.
    • Download the log file for further analysis.

Enabling logging at debug level is a helpful way to identify issues and understand what is happening within the system.

Diagnostics​

PandaGarage gives you the option to export a so called diagnostics zip file. This file includes all available logs and all your settings. This can be used to troubleshoot your settings or logs (for example if your PandaGarage is disconnecting from the garage door motor).

To download the diagnostics ZIP, go to Settings > Device and, in the Logging section, click the “Download Diagnostics” button.
