ESP32 S3 based controller for Hörmann Garage Doors.


Introduction​

If you want to go straight to the web installer, just press the button below. Otherwise, follow the getting started guide.




What is PandaGarage?​

PandaGarage is a control board designed for Hörmann garage doors, to make them smart. Using this board, you can control your garage door using the WebUI, Home Assistant, HTTP API or MQTT. Additionaly the board comes with three onboard sensor for temperature, humidity and a light sensor. It features I2C header pins for additional sensors. It also includes a buzzer, which enables you to play different melodys while the door opens or closes.


Where can I get this board?​

Currently, the board is in its final development phase and is not yet available for purchase. However, a preorder option is planned for the near future.

To stay updated, consider joining our Discord community.

The entire project is open-source, giving you the freedom to modify it as you like.


How can I support this project?​
